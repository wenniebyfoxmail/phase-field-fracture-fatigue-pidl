## Detailed entries

### F1 — N_f vs Umax: framework-level match

**Purpose**: Show that PIDL framework-level fatigue prediction agrees with FEM within ±10% across 5+ amplitudes (cross-Umax) AND across many methods at u=0.12 (cross-method). This is the headline §4.1 finding.

**Generator**: `plot_fig_F1_nf_vs_umax.py`

**Data sources**:
- FEM N_f (5 Umax): `~/Downloads/_pidl_handoff_v2/post_process/SENT_PIDL_{08,09,10,11,12}_timeseries.csv` last row N count + values from `docs/FEM.md` §3 table for u=0.13/0.14
- PIDL baseline (multiple Umax): local archives `hl_8_*N300*Umax<u>` + `_pidl_handoff_v2`
- PIDL Queue E soft-sym (Umax=0.10, 0.11, 0.12 (×3 seeds), 0.13): Windows-PIDL outbox + memory `finding_symmetry_prior_may6.md`
- PIDL other methods at u=0.12 (Williams, Enriched, spAlphaT, Oracle, Strac alone): outbox + memory tables

**Output panels**:
- (a) Log-y N_f vs Umax (FEM line + PIDL baseline + PIDL soft-sym), n=3 error bar at u=0.12
- (b) Horizontal bar chart of N_f at u=0.12 across 8 methods with ±10% band

**Paper § linkage**: §4.1 main argument that "PIDL framework-level prediction is robust to method, seed, Umax".

### F2 — V4+V7 cross-method bar chart

**Purpose**: Show the §4.2.1 "two empirically decoupled failure modes" finding. 13 PIDL configurations + FEM reference compared on V4 (mirror symmetry RMS) and V7 (BC residual). Visualises the "decoupling" empirically.

**Generator**: `plot_fig_F2_v4_v7_barchart.py`

**Data source**: `SENS_tensile/validation_v4_v7_all_methods.csv` (generated by `summarize_v4_v7_all_methods.py` from individual `validation_report.json` files).

**Output panels**:
- (a) V4 mirror RMS — log scale, sorted ascending, PASS line at 2e-4 (Mandal et al. 2019), "0 by construction" annotation for A1-bearing rows
- (b) V7 σ_xx relative residual — log scale, sorted ascending, PASS/WARN/FAIL bands, FEM-8 ref line at 0.12%

**Paper § linkage**: §4.2.1 main figure. Embeds the 13-method table that anchors the "targeted supervised regularization" argument.

## Style conventions

- Colours: FEM = black; PIDL baseline = `#0B4992` (blue); PIDL soft-sym = `#0B7A0B` (green); Strac alone = `#C0282A` (red); Combo = `#0B7A0B` (green)
- Markers: FEM = circle; PIDL = square (interventional) or triangle (baseline)
- Hollow markers (white fill + coloured edge) for emphasis
- Log axes when value span > 1 decade
- PASS/WARN/FAIL bands: green / orange / red with low alpha
- Embedded annotations for key callouts (e.g. "n=3 seeds", "bimodal spike")
